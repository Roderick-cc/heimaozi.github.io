### 1.函数防抖

> ​	lodash 是第三方JavaScript工具函数库

原理：使用闭包，每次调用的时候把上一次的定时器清除

```javascript
function debounce(callback,time){
  let timer
  return function(){
    //清除定时器
    window.clearTimeout(timer){
      timwe =window.setTimeout(function(){
        callback()
      },time)
    }
  }
}
```
